Windows CMD Commands Windows Command Line

C:> ping

Sends ICMP Echo Request messages to a specified network host to test connectivity and measure the response time.

C:> ipconfig

Displays the IP configuration for all network interfaces on the local machine, including IP addresses, subnet masks, and default gateways.

C:> ipconfig /all

Displays detailed IP configuration information for all network interfaces on the local machine.

C:> ipconfig /release

Releases the IP address of the specified network adapter.

C:> ipconfig /renew

Renews the IP address of the specified network adapter.

C:> pathping

Provides information about the path that network packets take to reach a destination, including information about packet loss and latency at each hop.

C:> tracert

Traces the route that packets take to reach a network host, showing the IP addresses of the intermediate hops.

C:> netstat

Displays active network connections and listening ports on the local machine.

C:> netdiag

Performs a variety of network diagnostic tests to diagnose common network connectivity issues.

C:> nbtstat -a

Displays NetBIOS over TCP/IP (NetBT) protocol statistics and current TCP/IP connections using NetBIOS.

C:> hostname

Displays the host name of the current computer.

C:> arp -a

Displays the Address Resolution Protocol (ARP) cache, which maps IP addresses to MAC addresses.

C:> nslookup

Performs DNS (Domain Name System) queries to retrieve information about domain names, IP addresses, and other DNS records.
